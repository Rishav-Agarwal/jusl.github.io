Object Oriented Systems


This course is meant to teach the concept of Object Oriented Systems. Concepts of Object Oriented Programming, UML diagrams and Design Patterns are taught in this course. Programming is done in Java Language. BoUML software is used for UML Diagram.

Professor: Dr. Samiran Chattopadhyay


Lecture Slides:

  1. Java Classes
  2. Java Inheritance
  3. Java IO
  4. UML Intro and Usecase
  5. UML Class Interaction Diagrams
  6. UML State Chart Diagram
  7. Design Patterns

Assignments:

  1. OOS
  2. Threads
  3. UML
  4. Design Pattern
